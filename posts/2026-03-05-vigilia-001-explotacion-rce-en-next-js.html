**1. Resumen Ejecutivo**

El presente documento detalla los hallazgos técnicos derivados del análisis forense realizado tras la detección de una intrusión activa en el entorno de aplicación objetivo. Se ha confirmado la explotación exitosa de una vulnerabilidad de tipo Prototype Pollution, la cual permitió a un actor adversario modificar la cadena de prototipos nativa del objeto JavaScript global. Esta manipulación facilitó la ejecución remota de código arbitrario (RCE) mediante la inyección de comandos en el módulo child_process.

**2. Análisis Técnico de la Vulnerabilidad**

La explotación se originó a través de un endpoint de API que procesaba datos JSON no validados provenientes del cliente. El atacante envió una estructura de objeto maliciosa diseñada para sobrescribir propiedades críticas en Object.prototype, específicamente apuntando a funciones de deserialización o configuración interna. Al modificar el prototipo raíz, se alteró el comportamiento esperado de las librerías dependientes, permitiendo la ejecución de lógica no autorizada.

Se identificó que la cadena de ataque culminó con una llamada al módulo child_process, un componente del entorno Node.js utilizado para ejecutar comandos del sistema operativo subyacente. El payload fue ofuscado mediante codificación Base64 para evadir los filtros de seguridad perimetrales y los sistemas de detección de intrusiones (IDS) basados en firmas estáticas.

**3. Infraestructura de Comando y Control (C2)**

Durante la fase de ejecución, el sistema comprometido estableció una conexión saliente hacia la dirección IP 104.236.195.25. El tráfico de red capturado muestra patrones consistentes con un canal de comunicación C2 persistente. Los paquetes analizados contienen datos cifrados que corresponden a instrucciones de exfiltración y mantenimiento de acceso. Se ha procedido al bloqueo inmediato de esta dirección en el firewall perimetral y se han notificado los registros de DNS para su mitigación global.

**4. Recomendaciones Inmediatas**

**5. Conclusión**

El incidente representa una amenaza significativa a la integridad y confidencialidad del sistema. La combinación de Prototype Pollution con inyección de comandos del sistema indica un actor sofisticado con capacidad para escalar privilegios. Se requiere una revisión completa de la arquitectura de seguridad antes de restablecer el servicio completo.
